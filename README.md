# HP ZBook Firefly G7 — Interactive Product Landing Page

An immersive, Apple-inspired product landing page built with **React.js**, **GSAP**, and **ScrollTrigger**. The experience transforms scrolling into a cinematic product showcase by synchronizing a high-quality product video with the user's scroll position.

## Features

### Scroll-Synchronized Video

The hero section features a product video whose playback is controlled entirely by the user's scroll. As visitors scroll, the video advances frame by frame, creating an immersive and interactive storytelling experience.

### Interactive Product Showcase

The landing page presents the HP ZBook Firefly G7 through a sequence of cinematic animations, including:

* Hero introduction
* Product rotations
* Hardware close-ups
* Internal component reveal
* Feature highlights
* Smooth transitions between sections
* Final product call-to-action

### GSAP Animations

Animations are powered by GSAP and ScrollTrigger, including:

* Scroll-linked timelines
* Pinned sections
* Fade and reveal effects
* Scale and transform animations
* Smooth text transitions
* Layered motion effects

### Responsive Experience

Designed to deliver a consistent experience across desktop, tablet, and mobile devices while maintaining smooth animations and optimized performance.

## Installation

Clone the repository.

```bash
git clone https://github.com/your-username/hp-zbook-landing.git
```

Navigate into the project.

```bash
cd hp-zbook-landing
```

Install dependencies.

```bash
npm install
```

Run the development server.

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

> If you're using Create React App instead of Vite, use `npm start` and open `http://localhost:3000`.

---

## Animation Flow

1. The landing page opens with a fullscreen hero section.
2. The hero is pinned during scrolling.
3. The product video is synchronized with the user's scroll position.
4. Feature content appears in sync with the animation.
5. The experience concludes with product specifications and a call-to-action.
